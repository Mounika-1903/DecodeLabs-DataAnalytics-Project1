## Project 1: Data Cleaning & Preparation

### Objective
Clean a raw e-commerce dataset by handling missing values, duplicate records, and incorrect data formats.

### Tools Used
- Microsoft Excel

### Dataset
- Rows: 1201
- Columns: 14

### Tasks Performed
- Removed duplicate records
- Handled missing values
- Standardized date formats
- Corrected text formatting
- Validated numeric fields
- Verified TotalPrice calculations

### Final Outcome
The dataset was cleaned successfully and is ready for analysis, dashboards, and reporting.
